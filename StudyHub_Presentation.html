INTRODUCTION
PROJECT VISION
  • Build a full-stack real-time collaborative study platform
  • Help students form study groups and share resources
  • Enable live peer-to-peer communication and note-sharing
MOTIVATION
  • Students lack a unified tool for study collaboration
  • Existing platforms don't combine notes + chat + files
  • Remote learning needs real-time coordination tools
KEY FEATURES
  • Study groups with live chat & member management
  • Collaborative notes with auto-save to database
  • File/resource library with upvotes & comments
  • Friend system with real-time notifications
  • Pomodoro focus timer
  • Meeting link sharing within groups
3 / 10
PROBLEM STATEMENT & OBJECTIVES
PROBLEM STATEMENT
  • No single platform for group-based study collaboration
  • Notes and files scattered across multiple apps
  • Lack of real-time communication during study sessions
  • No way to track shared resources within a study group
  • Difficult to coordinate meeting times and links
KEY OBJECTIVES
  • Create persistent, DB-backed collaborative notes
  • Provide real-time group chat powered by Socket.io
  • Build a friend system with instant notifications
  • Enable file uploads with group-level visibility
  • Secure all data with JWT-based authentication
  • Ensure auto-save, live updates, and push notifications
4 / 10

CHALLENGES & FUTURE SCOPE
CHALLENGES FACED
  • MongoDB Atlas DNS resolution on certain ISPs (fixed via Google DNS override)
  • Stale closure bugs in debounced auto-save (solved with useRef)
  • Socket.io room management for multi-group users
  • Merging live DB notes with local seed data cleanly
  • CORS & JWT token propagation with Socket.io handshake
FUTURE SCOPE
  • AI-powered note summarisation using an LLM API
  • Video/audio study rooms (WebRTC integration)
  • Mobile app (React Native)
  • Export notes as PDF or Notion pages
  • Offline mode with background sync
  • Gamification — streaks, badges, leaderboards
